# Day 03 - Given an Array, Return the count of Even Numbers and store it in an array

## Problem Statement

Given an array of numbers, write a function `countEvens()` that returns the count of even numbers in the array.

- Input must be an array of finite numbers.
- Count only even integers
- Return false for non-array inputs
- Return false for array containing non-numeric values
- Return false for arrays containing infinite or NaN values
- The value 0 counts as an even number.
