# Day 03 - Given an array of numbers, return the smallest number in the array.

## Problem Statement

Create a function `findSmallestNumber(arr)` that takes an array of numbers as input and returns the smallest number in that array.

- Input must be an array of finite numbers.
- If the array is empty, return `null`.
- Return false for non-array inputs.
- Return false for array containing NaN, Infinity, or -Infinity
- Return false for non-array inputs.
